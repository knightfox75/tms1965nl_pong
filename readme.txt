TMS1965NL PONG

1. ¿Qué es este proyecto?
El objetivo de este proyecto es la construcción desde cero de una videoconsola de primera generación basada en el PONG-IN-A-CHIP TMS1965NL de Texas Instruments.

2. ¿Que se suministra?
Todos los diagramas y serigrafías de muestra para su construcción.
